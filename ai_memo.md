# AI备忘录

## 摄像头摆放
- 摄像头型号：IMX219
- 摆放建议：请参考 CAMERA_ORIENTATION.md 文件，确保左右摄像头对齐，避免倾斜。
- 校准数据存放路径：calib_data/ 及各项目下的 calib_data/ 目录。

## 远程使用说明
- 通常通过 SSH 远程连接 Jetson Orin Nano Super 设备。
- 设备桌面环境通常关闭，仅使用命令行进行操作。
- 推荐使用 VS Code Remote 或直接使用终端进行开发和调试。
